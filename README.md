## Install

[npm][]:

```sh
npm install @ladjs/pug-lint-config-lad
```

[yarn][]:

```sh
yarn add @ladjs/pug-lint-config-lad
```


## Usage

```js
// inside .pug-lintrc.js
module.exports = {
  extends: '@ladjs/pug-lint-config-lad',
  // your configs
};
```
